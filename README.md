본 프로젝트는 C++14를 사용하여 구현한 도서 대여 시뮬레이션 시스템이다. 실무 수준의 객체지향 설계, 디자인 패턴, 메모리 제어의 체득을 목표로 한다.

## 주요 특징  
- SOLID 원칙에 기반하여 클래스를 역할 단위로 나눠 약 60개 이상으로 구성
- 커맨트 패턴, 전략 패턴, 상태 패턴, 옵저버 패턴, 팩토리 패턴 등 디자인 패턴들 적용
- 명령 객체, 우선순위 큐, 커스텀 정렬, FSM을 이용한 다단계 입력 처리 등 고급 기능 구현
- 사용자 정의 예외 계층과 SafeExecutor를 사용한 예외 안정성 증진
- 모든 동적 객체는 스마트 포인터를 쓰지 않고 직접 메모리 할당 및 해제를 하여 자원 수명 제어 연습
- 클래스 간 의존성 최소화를 위해 인터페이스와 의존성 주입 구조 적용
- 사용자 입력을 정규표현식을 사용한 FSM으로 처리하여 유효성 검사
- MVC 형식 UI 구조: View, Controller, Router, Session, AppState 등으로 구현
- 모든 사용자 행동은 로그와 이벤트 디스패치를 통해 추적 가능
- 도서 정보는 실제 도서관 데이터셋 활용(출처: https://www.data.go.kr/data/3072689/fileData.do), 프로젝트에 맞게 정제


## 데이터셋 정제 과정(정제 코드는 'clean' 브랜치에 저장)
1. '대여료' 칼럼 추가(대여료는 1,000~10,000의 정수 난수)  
2. 한자 및 일본어가 포함된 레코드는 제거  
3. 무작위로 1,000개 레코드 추출  
4. '출판년' 칼럼에서 숫자만 추출
