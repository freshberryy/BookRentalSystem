본 프로젝트는 C++14를 사용하여 구현한 도서 대여 시뮬레이션 시스템이다. 실무 수준의 객체지향 설계와 구현을 목표로 한다.

## 주요 특징  
- 클래스는 책임 단위로 분리하여 약 30개로 구성
- 모든 동적 객체는 스마트 포인터를 쓰지 않고 직접 메모리 할당 및 해제를 하여 자원 수명 제어 연습  
- 실제 도서관 데이터셋 활용(출처: https://www.data.go.kr/data/3072689/fileData.do), 프로젝트에 맞게 정제
- 데이터셋 정제 과정(정제 코드는 'clean' 브랜치에 저장)
  1. '대여료' 칼럼 추가(대여료는 1,000~10,000의 정수 난수)
  2. 한자 및 일본어가 포함된 레코드는 제거
  3. 무작위로 1,000개 레코드 추출
  4. '출판년' 칼럼에서 숫자만 추출
- csv 파일 역직렬화 구현
- 도서 정보, 대여 정보를 csv 파일로 직렬화 구현
- 사용자의 모든 행동은 로그로 남도록 구현

## 주요 기능
- 도서 등록, 검색, 대여, 반납, 삭제 등 기본적 CRUD
- 다양한 조건으로 도서 필터링 및 정렬
- 고객/관리자 계정 분리, 로그인 구현
- 대여 이력 저장 및 조회
- 도서 재고 수량 관리
- books.csv, users.csv를 일종의 DB처럼 활용 가능. 즉, 저장과 불러오기가 가능
- 다양한 상황(도서 중복, 로그인 실패, 재고 부족 등) 가정, 이러한 상황 발생 시 사용자 정의 예외 처리
